Projects

ArUbot project

The aim of this project was to build a fleet of robots detected by ArUco markers through an external camera.

During this group project, we modeled the robot using the OnShape software. Then, we constructed these robots. They are equipped with an ESP32 microcontroller and an L298N driver to control 2 motors.

My contribution to the project was the implementation of code on the microcontroller using the ArduinoIDE (integrated development environment). This IDE uses the C++ programming language, which is oriented towards hardware programming. I also developed the website for controlling the robots, for which I used the HTML programming language.
