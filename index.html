Projets

Le projet ArUbot

Le but de ce projet a été la construction d'une flotte de robots détectés par des marqueurs ArUco via une caméra externe.

Lors de ce projet de groupe, nous avons modélisé le robot grâce au logiciel OnShape. Ensuite nous avons construit ces robots. Ceux-ci sont équipés d'un microcontroleur ESP32, d'un driver L298N permettant de contôler 2 moteurs.

Ma contribution au projet a été l'implémentation de code sur le microcontrôleur via l'IDE (integrated development environment) ArduinoIDE. Cet IDE utilise le langage de programmation C++ qui est orienté dans la programmation de l'hardware. J'ai aussi développé le site web permettant le contrôle des robots. Pour cela j'ai utilisé le langage de programmation HTML.
